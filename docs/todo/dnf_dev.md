# 废弃监狱七日夜 - 游戏设计文档

## 一、概述与背景设定

### 1.1 综合背景
1940年，你们作为流民无处可归即将饿死，一位善良的A国军官以普通罪犯名义将你们抓捕，安顿在废弃监狱中，承诺七天后给予A国国民认证。一路上人员不断减少（从60多人减至48人），你们因战争麻木，以为只是走散。傍晚，你们抵达这座因不明原因废弃的监狱，获得片刻安宁。

### 1.2 隐秘背景（里线真相）
监狱地下深处埋藏着一颗来自异世界的陨石（下界之星）。科学家曾从中提取特殊细胞，以犯人为实验材料。实验失败后，诡异气息弥漫监狱，使其逐渐变得虚幻并消失于世界。每年特定时期，监狱会被拉回现实，吸引误入者成为其一部分。玩家进入的是幻境轮回，扮演被押送进监狱的流民（其中混有真正罪犯），因陨石力量失去记忆，完全代入犯人身份。最终需找到陨石并接触，才能终结循环。

### 1.3 前情提要
陨石自异空间坠落，当地政府建立监狱进行实验，实验意外使整个监狱被吞噬成异空间，只在特定时间段开放。陨石主人委托列车将陨石带回，新轮回开始时列车停在监狱门口。这座监狱以“和平”闻名，很少暴动，但那些没有离开的人实际被监狱吸收。

## 二、角色与人员配置

总人数：48人

| 角色类型 | 人数 | 功能说明 |
|---------|------|----------|
| 狂魔 | 2 | 白天旁观模式；夜晚在自己房间变为实体，拥有**速度2**。被枪杀后会眩晕一个晚上（建议用录像摄像头监控）。 |
| 军人 | 2 | 拥有警棍和手枪。手枪**一天只有一发子弹**，白天不可枪杀别人（否则降级为罪犯）。投票完成后警察对被票人有高亮，此时警棍可两下敲死罪犯。 |
| 杀手 | 4 | 开局与普通罪犯无差别，需完成任务。**夜晚觉醒**：有一把刀（CD 50秒，一晚上只能开三次）和一根撬棍（仅使用一次）。杀掉一人后可申请援助，第二天晚上获得开锁器（3次）或手枪（一次性、可隔玻璃杀人）。无需吃饭，可啃食尸体。 |
| 厨师 | 1 | 白天在食堂工作，为所有人制作食物并检查水源。需警惕食物被掉包。设置专门食物箱子。食物种类：玉米糊糊、黑面包、肉。检查水源需占用10个食物位。厨师最多做40个食物，食堂初始自带50份食物。厨师工作可恢复san值。 |
| 毒师 | 1 | 可制作被污染的食物（肉、黑面包、玉米糊糊）。第一天可做5个，之后每天只能做1个。杀死5人后解锁污染水源选项。污染食物需在夜里放入食物箱子。 |
| 心理学大师 | 2 | 可以以扣80% san值为代价，观察一个人是否杀过人（PUA诈骗类型）。 |
| 锁匠 | 4 | 可用25% san值制作一个维修工具，修复被撬锁损坏的门（只能修复由撬锁器造成的损坏）。 |
| 普通罪犯 | 34 | 白天需吃饭、听讲会、上厕所。发现尸体可报告进入公告会，拥有投票权。 |

## 三、游戏规则

### 3.1 整体规则
- 每完成一个任务可恢复40% san值。
- 吃饭是刚需：每人每天必须吃上一口食物，否则san值不会回复。

### 3.2 食堂牌子规则
1. 登入监狱时为下午，第一天早上只有2分钟，需快速吃饭。
2. 食物有限，请勿浪费。
3. 夜晚食堂相对安全，无处可去可来食堂过夜。
4. 食堂食物不一定安全。
5. 夜里保持安静，否则有东西会盯上你。

### 3.3 宿舍楼牌子规则
1. 在自己房间相对安全，不要随便窜门。
2. 宿舍楼夜里有怪物出没，请勿出门。
3. 保持80%以上 san值，否则房间将不认识你。
4. 看到尸体请报告。
5. 夜里保持安静，否则有东西会盯上你。

### 3.4 警察宿舍规则
1. 保证人员安全，枪是最后的保障，而非制裁手段。
2. 怪物白天会死亡，需找出流民中藏匿的吃人者。
3. 恶魔无法杀死，但枪可以暂时驱逐。
4. 你们永远是安全的，只要不出门。
5. 夜里会有被恶魔蛊惑的人寻求血肉滋养，凭良心决定是否拯救。

## 五、任务与San值系统

- **初始san值**未明确，但需保持80%以上避免房间排斥。
- **恢复方式**：
  - 每完成一个任务：+40% san值
  - 厨师工作：恢复san
- **消耗方式**：
  - 心理学大师观察他人：-80% san值
  - 锁匠制作维修工具：-25% san值
- **负面效果**：
  - san值低于80%：房间不认识你（无法进入或会被驱逐？）
  - 不吃饭则san值不回复

## 六、游戏流程

### 6.1 总体流程（表线）
- 玩家进入监狱，被告知一周后监狱需一批尸体，典狱长决定在食物中随机下毒。存活一周的人将被释放并获得封口费，不配合者饿死或冻死。
- 第一夜所有牢房门被打开，白天敞开的办公室全部上锁。玩家发现各处刻下的规则，需遵守才能生存。
- 玩家行为可能包括：试图逃跑、贿赂典狱长、提前杀死他人等。

### 6.2 里线进程
- 监狱大门封闭且无法踏出，典狱长办公室无人，只有前人警示。
- 随着探索，狂魔秘密被解开，看守视线被蒙蔽。
- 幸存者前往地下实验室，找到一切源头——陨石（下界之星）。触摸陨石后幻境消散，列车出现在空地上。

## 七、结局设定

| 结局 | 条件 | 描述 |
|------|------|------|
| **团灭结局** | 所有人死亡 | 尸体被监狱吸收，成为其一部分。循环提前结束，全员迷失（返回列车）。 |
| **存活结局** | 找出危险人物，学会生存方法，打开监狱大门逃出 | 成功逃出后恢复记忆，但又被送回监狱（？待确认）。 |
| **真结局** | 收集前人信息，抵达地下实验室，接触下界之星 | 监狱在阳光中消散，一切终结。 |

## 八、备注

- 游戏时间：7天轮回，每天需管理食物、san值、夜间怪物等。
- 狂魔、军人等NPC行为受规则约束，玩家需在混乱中合作或对抗。
- 侦查与谎言机制：心理学大师可验杀人者，投票与高亮机制辅助指认。
- 资源管理：食物总量有限（初始50+厨师每日最多40），且可能被下毒。
